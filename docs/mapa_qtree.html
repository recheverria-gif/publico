  • QTREE - Árbol de Llamados

    Sistema para analizar referencias entre programas y archivos en IBM i, generando una vista jerárquica del flujo de llamados en TREETXT.

    • QTREE.ARBOL

      Define el comando de entrada. Recibe TOPGM, WTHFILE, WTHSRV y WTHUPD para controlar el análisis.

    • QTREE.ARBOL00CL

      Programa de instalación. Compila ARBOL01CL, ARBOL02CL y crea el comando ARBOL.

    • QTREE.ARBOL01CL

      Controlador principal. Limpia QTEMP, crea archivos auxiliares, llama a ARBOL02CL y muestra TREETXT al final.

      • QTREE.ARBOL02CL

        Motor del análisis. Ejecuta DSPPGMREF, separa programas y archivos, filtra datos con SQL, prepara overrides y llama al RPG generador.

        • QTREE.F_PGM_GRP

          Copia todas las referencias de programas desde QTEMP/REFPASO para su agrupación base.

        • QTREE.F_PGM

          Filtra únicamente objetos de tipo *PGM.

        • QTREE.F_PGM_SRV

          Filtra objetos de tipo *PGM y *SRVPGM cuando se solicita incluir programas de servicio.

        • QTREE.FILTROFIL

          Copia o agrupa las referencias de archivos para el procesamiento visual posterior.

        • QTREE.ARBOLRPG

          Genera la salida visual del árbol. Dibuja conectores, cabeceras, archivos usados y profundiza el análisis llamando nuevamente a ARBOL02CL.

          • QTREE.ARBOLRPGX

            Variante de ARBOLRPG con diferencias menores de presentación y comportamiento auxiliar.

          • QTREE.TREEFILE

            DDS del archivo físico TREEFILE. Guarda nombre del archivo y tipo de uso para control y consulta.
