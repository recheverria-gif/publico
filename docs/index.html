QTREE - Árbol de llamados para IBM i

QTREE es una utilidad que analiza referencias entre programas y archivos para construir una vista jerárquica del flujo de llamados. Este sitio reúne una portada de navegación, un mapa mental interactivo y una página técnica con la explicación de cada fuente.

Ver mapa interactivo Ver documentación técnica

Objetivo del sistema

El objetivo de QTREE es mostrar el árbol de llamados de un programa IBM i, permitiendo incluir archivos utilizados y, opcionalmente, programas de servicio. El resultado final se genera en el archivo temporal TREETXT y se presenta como una vista estructurada del flujo de ejecución.

Flujo principal

El proceso parte desde un comando de usuario, pasa por programas CL que preparan el análisis, ejecuta extracción de referencias mediante DSPPGMREF, filtra resultados con SQL y finalmente utiliza RPG para dibujar la salida del árbol.

Componentes principales

CMD

QTREE.ARBOL

Define el comando de entrada del sistema y sus parámetros de análisis.

CL

QTREE.ARBOL00CL

Compila los CL y crea el comando para dejar la herramienta instalada.

CL

QTREE.ARBOL01CL

Prepara temporales, llama al motor del análisis y muestra la salida final.

CL

QTREE.ARBOL02CL

Extrae referencias, filtra programas y archivos, y prepara los datos para RPG.

RPG

QTREE.ARBOLRPG / ARBOLRPGX

Dibuja el árbol de llamados y la lista de archivos usados en formato visual.

SQL / DDS

F_PGM, F_PGM_SRV, FILTROFIL, TREEFILE

Definen filtros auxiliares y archivos físicos usados durante el proceso.
